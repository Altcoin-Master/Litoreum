● Smartnodes need to lock some LTRM in your wallet to get rewards, which is different from general network nodes.At the same time, we have not enabled PoS Ban at present. Generally speaking, when your node is disconnected, you will not be able to get rewards. Yes, we have not enabled this verification mechanism, which means that you do not need to purchase an expensive VPS for configuration. You only need to lock LTRM in your wallet to get rewards. In the future, the node rewards of LTRM will be enough to cover the VPS fee , we will enable it.

● Now you can skip the VPS configuration directly and set up a virtual node.
https://github.com/Altcoin-Master/Litoreum/tree/main/doc/smartnodes#local-wallet-setup

● But I still hope that people with idle VPS can build a real node according to the following steps and open port 21001, all of which are voluntary.
https://github.com/Altcoin-Master/Litoreum/tree/main/doc/smartnodes

● As the litoreum network matures Smartnode collateral ramps up and you will need to recreate your Smartnode in order to increase the collateral amount.
